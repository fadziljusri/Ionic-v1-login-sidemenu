# Ionic-v1-login-sidemenu
An Ionic v1 mobile app template (login + sidemenu)

This project uses [Ionic Framework](http://ionicframework.com/).

## Install

    $ npm install
    $ ionic platform add android@6.1.2
    
## Run using web

    $ ionic serve --lab 
    
## Build android

    $ ionic build android 

## Run using USB debugging

    $ ionic run android
